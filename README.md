# Customer-Churn-Analytics
**Overview**
This project presents an interactive dashboard designed to analyze customer churn, understand churn drivers, and support data-driven retention strategies. The dashboard consolidates key churn metrics, customer segmentation insights, and visual analytics into a single, easy-to-use interface.
